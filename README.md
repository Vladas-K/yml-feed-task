# YML Feed Test Task

Решение тестового задания на Python/Django по формированию YML-фида для Яндекс Директа.

## Структура проекта

```text
test_task/
├── feed_task.py
├── test_feed_task.py
├── django_example.py
├── requirements.txt
├── .gitignore
└── README.md
```

## Требования

* Python 3.9+
* pytest

## Создание виртуального окружения

```bash
python -m venv venv
```

### Активация

Windows:

```bash
venv\Scripts\activate
```

Linux / macOS:

```bash
source venv/bin/activate
```

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Запуск тестов

```bash
pytest -v
```

## Реализовано

* Формирование корректного XML через `xml.etree.ElementTree`.
* Формат даты `YYYY-MM-DD HH:MM`.
* Фильтрация товаров по условиям задания:

  * товар активен;
  * категория активна;
  * название не пустое;
  * цена больше нуля;
  * указано корректное изображение (`http://` или `https://`).
* Сортировка товаров по `id`.
* Выгрузка только используемых категорий без дубликатов.
* Корректная обработка `available="true"` и `available="false"`.
* Вывод `oldprice` только если старая цена больше текущей.
* Исключение пустого элемента `<description>`.
* Безопасная обработка специальных символов (`&`, `<`, `>`, `"`).

## Тестирование

Автоматические тесты покрывают:

* корректность XML;
* формат даты;
* фильтрацию товаров;
* порядок категорий;
* порядок предложений;
* атрибут `available`;
* формат цен;
* правила вывода `oldprice`;
* отсутствие пустых описаний;
* обработку специальных символов.
